# Personal Digital Twin Dashboard

This project is a Level 3 submission for the LifeAtlas LPI Developer Kit (Design & UX Track).

## Overview
This is a UI/UX design of a personal digital twin dashboard that tracks health and productivity metrics such as sleep, exercise, nutrition, and stress.

## Features
- Health statistics overview
- Habit tracking
- Insights generation
- Actionable recommendations
